Key Features

  • Ultra-Realistic 3D Appearance: The button appears to protrude from the screen with a massive convex shape
  • Translucent Glass Material: Realistic index of refraction creates optical distortion and background blurring
  • Jelly-Like Physics: Realistic deformation with elastic spring forces and wave-like ripple effects
  • Advanced Lighting: PBR materials with highlights, shadows, and depth of field for photorealistic rendering
  • Interactive Controls: Real-time adjustment of material properties through intuitive sliders
  • Responsive Design: Optimized for all screen sizes with touch support for mobile devices
  • 60 FPS Performance: Smooth animation with optimized particle system and rendering
  • No Dependencies: Pure vanilla JavaScript and HTML5 Canvas - no external libraries required

How to Use

Basic Interaction:

  1. Click and hold on the button to activate deformation
  2. Drag while holding to increase deformation force
  3. Release to see the button spring back naturally
  4. Adjust sliders to change material properties in real-time

Parameter Effects:

  • Refraction Index: Higher values create more dramatic background distortion
  • Transparency: Controls overall opacity and glass-like appearance
  • Elasticity: Affects how quickly the button deforms and recovers
